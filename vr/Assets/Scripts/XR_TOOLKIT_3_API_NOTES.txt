=====================================
XR INTERACTION TOOLKIT 3.x API NOTES
=====================================

IMPORTANT: XR Interaction Toolkit 3.x changed several API names from version 2.x

FIXED COMPILATION ERRORS:
-------------------------

1. Event Arguments Changed
   OLD (v2.x): InteractableSelectEnterEventArgs
   NEW (v3.x): SelectEnterEventArgs
   
   OLD (v2.x): InteractableSelectExitEventArgs  
   NEW (v3.x): SelectExitEventArgs

2. Required Using Statements
   For XR Interaction Toolkit 3.x, you need:
   
   using UnityEngine.XR.Interaction.Toolkit;
   using UnityEngine.XR.Interaction.Toolkit.Interactables;
   using UnityEngine.XR.Interaction.Toolkit.Interactors;

3. Event Handler Signatures
   
   CORRECT:
   private void OnGrabbed(SelectEnterEventArgs args)
   {
       // Your code
   }
   
   INCORRECT:
   private void OnGrabbed(InteractableSelectEnterEventArgs args)
   {
       // Your code
   }


UNITY 6 PHYSICS API CHANGES:
-----------------------------

Unity 6 renamed several Rigidbody properties:

1. Damping Properties
   OLD: rb.drag
   NEW: rb.linearDamping
   
   OLD: rb.angularDrag
   NEW: rb.angularDamping

2. Velocity Properties
   OLD: rb.velocity
   NEW: rb.linearVelocity
   
   (angularVelocity remains the same)

3. These are already correctly implemented in the branch scripts!

TROUBLESHOOTING:
----------------

If you still see errors about "InteractableSelectEnterEventArgs":
→ Make sure you saved all files
→ Right-click in Unity Project window → Reimport All
→ Check that XR Interaction Toolkit package is version 3.x

If you see errors about "linearDamping":
→ Verify you're using Unity 6.x (not Unity 2021/2022)
→ Update to latest Unity 6 if needed
